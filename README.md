# VentaExpress---Proyecto1

### "VentaExpress" | Proyecto Sprint-1 | TripleTen

Este repositorio contiene datos de la empresa "VentaExpress", en formato XLSX, para el proyecto de Limpieza, Organizacion, Resumen y generacion de Insights de negocio, como analista de datos junior.

El Proyecto forma parte del curso de formacion Data Analyst de TripleTen

### INFORMACION DE LOS DATOS :

VentaExpress - Datos Originales - Proyecto 1.xlsx

### PARA VERIFICAR EL ANALISIS : 
Hojas en el Archivo :

"Analisis" : Metricas principales.
"Visualizaciones" : Ventas por ciudad y por mes, por el periodo analizado.
"Informe ejecutivo" : Insights de negocio.

### CONTEXTO DEL NEGOCIO :

Venta de productos de tecnología (laptops, teléfonos, auriculares, tablets) a través de su plataforma online en distintas ciudades latino americanas.

Solicitaron analizar datos de ventas por el periodo de Octubre - Diciembre 2024.
